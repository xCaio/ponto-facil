# PONTO FÁCIL

`TECNOLOGIA - Análise e Desenvolvimento de Sistemas`

`DISCIPLINA`

`4° Eixo`

Este projeto visa desenvolver um aplicativo de controle de ponto eletrônico que permitirá às empresas monitorar, registrar e gerenciar as horas trabalhadas de seus colaboradores de forma automática e eficiente. O sistema também fornecerá relatórios detalhados, ajudando na tomada de decisões e garantindo o cumprimento das obrigações legais e contratuais.
